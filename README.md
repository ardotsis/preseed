# preseed.cfg for Debian
>
> [!NOTE]
> My configuration is available on [https://get.ardotsis.com/preseed.cfg](https://get.ardotsis.com/preseed.cfg).

## Feature

- Run [dotfiles installer](https://get.ardotsis.com/df) automatically after OS installation.

## Test on VirtualBox

- Disable `Unattended installation` option in a new VM creation window.
- Enable EFI (UEFI) mode in expert settings window.
- Set network adapter mode to `Bridge Mode`.
